## Page 2

### Structure

1. **1 Page** maximum (especially for freshers)
2. **1 Column** preferable
   * a. ATS scans don't always parse information from columns properly.
   * b. Is not followed by senior applicants, so difficult to follow for human eyes.

***

3. Try listing in **reverse chronological order** (within sections)
4. **Relative font sizes should be close.** It's not like you use 20 for title and 8 for description

#### WORK EXPERIENCE (Example)
* **Aten Ventures** (July-Sept 2020)
  * **STRATERGIC CONSULTANT**
  * Worked on quality market research and distribution channel analysis of **marketing technology**.
  * Also on building of a similar marketplace to connect startups with VCs and other potential investors, researched on **200+ startups**.
* **SKK Infra Pvt. Ltd.** (Udaipur, India - July 2019)
  * **ENTERPRISE RESOURCE PLANNER**
  * Worked on various tools required in enterprise resource planning of the company and managed them.

***

5. Each bullet should have max 1 or 2 lines. No non-integral lines.
6. The order of sections depends on your history and JD
7. **White spaces Laxman Rekha**: Inexperience | Bad design
8. Name it like 'Rahool_Jaen_Resume'

### Some Keywords for Different Roles

#### Strategy keywords
1. Developed a product vision and strategy
2. Created product personas
3. Built a product roadmap
4. Conducted a strategic review
5. Recommended strategic initiatives

#### UX / UI design keywords
1. Designed initial wireframes
2. Gathered UX feedback from users
3. Researched competitors' UX
4. Provided feedback to design team
5. Contributed to the development of a design system

#### Coding keywords
1. Wrote detailed user stories for engineering team
2. Made initial tech difficulty assessments for features
3. Negotiated timelines with Lead Engineer / CTO
4. Helped make engineering trade-off decisions
5. Built front-end / back-end features for app XYZ.

#### Digital marketing keywords
1. Created value props with marketing team
2. Provided feedback on copywriting
3. Coordinated launch with marketing team
4. Tested five acquisition channels
5. Scaled a channel to X number of users

#### Data analysis keywords
1. Set key success metrics
2. Measured metrics' improvement
3. Performed analysis on large datasets
4. Generated actionable insights
5. Summarised data findings

#### Leadership keywords
1. Lead a team
2. Aligned stakeholders
3. Managed a project
4. Took an initiative
